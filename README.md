# SITE WEB DYNAMIQUE EN PHP

## INCLUDE - REQUIRE

- inclut un autre fichier
- include : ne bloque pas le script si le fichier n'est pas trouvé
- require : bloque le script si le fichier n'est pas trouvé
- include_once : ne bloque pas le script si le fichier a déjà été chargé
- require_once : bloque le script si le fichier a déjà été chargé
